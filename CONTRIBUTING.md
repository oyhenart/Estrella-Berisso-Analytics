# Contributing y Uso del Contenido

## ⚠️ Aviso Legal Importante

Este repositorio es mi **portafolio personal** como video analista de fútbol.

### Propiedad del Contenido

- El trabajo detallado, datos y análisis pertenecen a **Estrella de Berisso** (club de fútbol amateur de La Plata, Argentina)
- Los jugadores juveniles (menores de edad) están **anonimizados** por protección de datos personales
- Este contenido se muestra únicamente con fines **educativos y de portafolio profesional**

### Términos de Uso

✅ **PERMITIDO:**
- Revisar el código con fines educativos
- Aprender de las metodologías y técnicas mostradas
- Contactarme para oportunidades laborales

❌ **NO PERMITIDO (sin permiso por escrito):**
- Usar el contenido para fines **comerciales**
- Reproducir, distribuir o modificar datos/tácticas del club
- Usar nombres, imágenes o datos de jugadores (aunque estén anonimizados)
- Revender o monetizar el contenido de este repositorio
