## 🔒 Política de Uso y Privacidad

### Propósito de este Repositorio

Este es mi **portafolio personal** como video analista de fútbol amateur.

### Contenido Sensible

Este repositorio contiene trabajo realizado para **Estrella de Berisso** (La Plata, Argentina):
- Análisis de rendimiento de equipo
- Datos de jugadores juveniles (menores de edad)
- Tácticas y estrategias del equipo

### Protecciones Implementadas

- ✅ **Nombres de jugadores juveniles están ANONIMIZADOS**
- ✅ **Datos sensibles han sido eliminados o reemplazados por datos dummy**
- ✅ **Videos/archivos originales no están incluidos**

### Uso Prohibido

**NO SE PERMITE** (sin autorización escrita):

1. **Uso comercial** del contenido de este repositorio
2. Distribución, reproducción o modificación de datos del club
3. Uso de información táctica para competir con Estrella de Berisso
4. Identificación de jugadores menores a partir de datos anonimizados
5. Revender o monetizar cualquier parte de este contenido
